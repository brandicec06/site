City + Code


City+Code envisions new contracts of generative urban design. Idiosyncratic methods of urban design methodologies were studied in this studio as a framework for generating new compositions of the built environment. The contemporary scope of architectural practice requires a wider scope of implementation for architectural design. Cities are being built as buildings were in prior centuries, however, architectural methodologies should inform such a scale of design practice. Emergent architectural designs must speak to higher order languages which will soon define a new paradigm in architectural practice.
