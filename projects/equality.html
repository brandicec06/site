Visualizing Income Equality


This project maps socioeconomic indicators to United States zip codes to illustrate wealth inequality in the united states. The application is built in the Processing software, with the base code used from Ben Fry's zipcode application.

At the zip code level, maximum and minimum values are denoted by the yellow and red marks, respectively. As different criteria are chosen from the left-hand panel, maximum and minimum values with in the currently zoomed window area are listed to the left. Directly below the "local" numbers are the national averages for the currently searched attribute, allowing quick comparisons to national averages.

Project Team: Brandon Cuffy, Wes Thomas, Nikhil Gahlawat
