Skill Evaluation Tool


Co-developer of application written in Processing to create a tool for Skills evaluation for over 600 employees at NBBJ. Using data fathered from surveys, data is now visualized and able to be sorted and filtered by employee location, title, software packages, and discipline. In addition, the Skills Tool will allow the balanced assembly of project teams with adequate skill sets needed for the execution of design projects from concept design through construction documentation.

An essential feature of the tool it the ability to create skill baselines for each role, ensuring employees are able to get effective training for software skills not at the baseline levels. Also, the tool allows employee training priorities to be recorded and saved as staff members are being reviewed. This will help more efficiently distribute annual budget allocations for training across the entire breadth of the firm.
